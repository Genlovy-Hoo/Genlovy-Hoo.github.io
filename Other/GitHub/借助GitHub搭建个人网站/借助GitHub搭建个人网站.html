# 借助GitHub搭建个人网站 1. 域名准备。如果想有一个自己喜欢的域名，可以去买一个（genlovy.cn是在阿里云买的）。 2. 创建GitHub账号并在GitHub新建一个Repository，Respository name格式为：yourname.github.io，yourname最好写成自己的GitHub账号名。 3. 进入Settings，GitHub Pages里面的Source选择master branch，然后在Source选项的上面就可以看到一个网址，这个地址就是目前个人网站的地址。 4. 添加文件。进入Repository主页，Upload files，如果上传的文件中有index.html，则访问上面提到的网址时会默认打开这个文件。 5. 将网址与自己买的域名绑定。首先去买域名的网站进行域名解析。在阿里云解析时记录类型填CNAME，主机记录填www，记录值填Github Pages页面上的网址，头部去掉（https去掉），然后保存解析。 6. 在GitHub上Repository的Settings中填写Custom damain为自己的域名（[www.genlovy.cn](http://www.genlovy.cn/)），Save。 7. 访问自己的个性域名（[www.genlovy.cn](http://www.genlovy.cn/)）就可以进入个人网站了。当然要保证域名有效（实名认证完成，没有过期）。 8. 说明：如果不需要设置个性域名的话，只通过2，3，4步就可以完成个人网站的创建了。
